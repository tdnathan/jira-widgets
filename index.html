Countdown Timer — Usage

Adding to Jira

  1. Go to your Jira dashboard and click Add gadget.
  2. Search for Webpage and add it (Jira Cloud may require a marketplace gadget).
  3. Paste the widget URL with your desired parameters into the URL field.
  4. Set the gadget height (recommended: 200–260px) and save.

URL Parameters

Param Required Description
title No Display title shown above the clock. Defaults to Countdown. Use + or %20 for spaces.
target No Target date/time in ISO 8601 format. e.g. 2027-12-29T09:00. If omitted, uses the last value set via the settings panel (stored in localStorage).

Examples

Basic — title + target
countdown-timer.html?title=Sprint+End&target=2026-04-15T17:00
Release countdown
countdown-timer.html?title=v2.0+Launch+In...&target=2026-09-01T00:00
Full GitHub Pages URL (for Jira Webpage gadget)
https://tdnathan.github.io/jira-widgets/countdown-timer.html?title=AJ2028+Begins+About...&target=2027-12-29T09:00
